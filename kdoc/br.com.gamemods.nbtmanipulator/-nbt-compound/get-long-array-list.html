getLongArrayList

fun getLongArrayList(name: String): NbtList<NbtLongArray>

Returns the NbtList of long arrays mapped to that name. The tag and it's values will be linked and any modification will also change the mapped value.

Exceptions

ClassCastException - If the NbtTag is not a NbtList

TypeCastException - If no value exists for that name

IllegalStateException - If the list is not empty and contains any tag with class different then NbtLongArray
