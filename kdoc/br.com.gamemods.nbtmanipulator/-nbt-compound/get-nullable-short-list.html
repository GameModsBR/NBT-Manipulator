getNullableShortList

fun getNullableShortList(name: String): NbtList<NbtShort>?

Returns the NbtList of shorts mapped to that name. The tag will be linked and any modification will also change the mapped value.

Will return null if no value is mapped or it is mapped to an other type tag.

Exceptions

IllegalStateException - If the list is not empty and contains any tag with class different then NbtShort
