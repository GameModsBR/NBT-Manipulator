getIntArrayList

fun getIntArrayList(name: String): NbtList<NbtIntArray>

Returns the NbtList of int arrays mapped to that name. The tag and it's value will be linked and any modification will also change the mapped value.

Exceptions

ClassCastException - If the NbtTag is not a NbtList

TypeCastException - If no value exists for that name

IllegalStateException - If the list is not empty and contains any tag with class different then NbtIntArray
