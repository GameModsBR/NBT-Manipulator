getNullableDoubleList

fun getNullableDoubleList(name: String): NbtList<NbtDouble>?

Returns the NbtList of doubles mapped to that name. The tag will be linked and any modification will also change the mapped value.

Will return null if no value is mapped or it is mapped to an other type tag.

Exceptions

IllegalStateException - If the list is not empty and contains any tag with class different then NbtDouble
